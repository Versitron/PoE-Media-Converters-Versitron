## Frequently Asked Questions

### What is a PoE media converter?  
A PoE media converter is a networking device that converts Ethernet signals to fiber while providing Power over Ethernet to connected devices like IP cameras or wireless access points.

### Why use PoE media converters with fiber?  
Fiber links allow extended distances and improved performance compared to copper-only PoE connections, especially for large campuses or distributed installations.

### Do PoE media converters require additional power?  
Yes — PoE media converters need a power source to deliver PoE to connected devices, typically via a local power adapter or centralized DC/AC power supply.
